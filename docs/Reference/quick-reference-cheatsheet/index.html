Common Code Patternsâ€‹

Create Documentâ€‹

using (var doc = WordprocessingDocument.Create(
    filepath, WordprocessingDocumentType.Document))
{
    var mainPart = doc.AddMainDocumentPart();
    mainPart.Document = new Document();
    var body = mainPart.Document.AppendChild(new Body());
    
    // Add content
    
    mainPart.Document.Save();
}

Add Paragraphâ€‹

var para = new Paragraph();
var run = new Run();
run.AppendChild(new Text("Hello World"));
para.AppendChild(run);
body.AppendChild(para);

Add Formatted Textâ€‹

var run = new Run();
var runProps = new RunProperties(
    new Bold(),
    new FontSize { Val = "28" },
    new Color { Val = "FF0000" }
);
run.AppendChild(runProps);
run.AppendChild(new Text("Bold Red Text"));

Create Table Cellâ€‹

var cell = new TableCell();
cell.AppendChild(new TableCellProperties(
    new TableCellWidth { Width = "3000", Type = TableWidthUnitValues.Dxa }
));
var para = new Paragraph(new Run(new Text("Cell content")));
cell.AppendChild(para);

Default Stylesâ€‹

  • Normal - Base paragraph style
  • Heading1 - 16pt bold, centered

Standard Measurementsâ€‹

  • Cell margins: 108 twips left/right, 0 top/bottom

Debugging Tipsâ€‹

Document Won't Openâ€‹

  1. Rename .docx to .zip
  2. Extract contents
  3. Check word/document.xml for XML errors
  4. Use OpenXML SDK tool

Images Missingâ€‹

  1. Check /word/media/ folder exists
  2. Check document.xml.rels has image relationships
  3. Verify relationship IDs match

Formatting Lostâ€‹

  1. Check styles.xml exists
  2. Verify style IDs match in document.xml
  3. Ensure w:pStyle or w:rStyle references valid style
